# SCP 25 Techniques — 기회를 없애면 범죄가 줄어든다

> Background and theory: Read references/foundation.md

---

## 25가지 기법 (5 x 5 매트릭스)

### 범주 1 — 노력 증가 (Increase the Effort)

| # | 기법 | 경찰 적용 예 |
|---|---|---|
| 1 | 표적 경화 | 상가 방범 점검 |
| 2 | 접근 통제 | 아파트 출입 통제 개선 |
| 3 | 출구 심사 | 마트·도서관 도난 방지 |
| 4 | 가해자 우회 | 시위대·반대진영 동선 분리 |
| 5 | 도구·무기 통제 | 편의점 주류 판매 ID 확인 |

### 범주 2 — 위험 증가 (Increase the Risks)

| # | 기법 | 경찰 적용 예 |
|---|---|---|
| 6 | 감시 확대 | CCTV 통합관제센터 |
| 7 | 자연감시 지원 | CPTED 조명 점검 |
| 8 | 익명성 감소 | 배달기사 인증 시스템 |
| 9 | 장소 관리자 활용 | Third-Party Policing |
| 10 | 공식 감시 강화 | 핫스팟 순찰 |

### 범주 3 — 보상 감소 (Reduce the Rewards)

| # | 기법 | 경찰 적용 예 |
|---|---|---|
| 11 | 대상 은닉 | "차량 내 귀중품 미보관" 캠페인 |
| 12 | 대상 제거 | 현금 없는 매장 확대 |
| 13 | 재산 식별 | 자전거 등록제 |
| 14 | 시장 교란 | 중고나라·당근 사기 대응 |
| 15 | 이익 거부 | 스마트폰 도난 시 원격 잠금 |

### 범주 4 — 도발 감소 (Reduce Provocations)

| # | 기법 | 경찰 적용 예 |
|---|---|---|
| 16 | 좌절 감소 | 민원실 동선·대기 개선 |
| 17 | 분쟁 회피 | 축구장 관중 분리 |
| 18 | 감정 자극 감소 | 술집 영업시간 제한 |
| 19 | 동료 압력 감소 | 청소년 음주 분리 |
| 20 | 모방 억제 | 자살 보도 가이드라인 적용 |

### 범주 5 — 변명 제거 (Remove Excuses)

| # | 기법 | 경찰 적용 예 |
|---|---|---|
| 21 | 규칙 설정 | 공원·시설 이용 규칙 명시 |
| 22 | 안내 게시 | CCTV 경고판 |
| 23 | 양심 환기 | "이 건물은 CCTV 감시 중" |
| 24 | 준수 지원 | 공공화장실 확충 |
| 25 | 약물·알코올 통제 | 심야 주류 판매 제한 |

## 적용법 — 거래처 사칭 사기를 SCP로 분석하면

| SCP 범주 | 적용 |
|---|---|
| 노력 증가 | 2인 재확인 프로토콜 → 사기범이 우회해야 할 단계 추가 |
| 위험 증가 | 발신번호 인증 시스템 → 변작 전화 탐지 |
| 보상 감소 | 지연 송금 규칙 → 즉시 인출 방지 |
| 도발 감소 | 해당 없음 (감정 범죄 아님) |
| 변명 제거 | "신규 계좌 입금 시 반드시 전화 확인" 규정 명문화 |

→ SARA의 Response 단계에서 SCP 25가지를 체크리스트로 쓰면 **"순찰 강화"를 넘어서는 개입 옵션**이 자동으로 나옵니다.

---

## 자주 하는 실수

1. **기법 1(표적 경화)에만 집착** — 나머지 24가지를 무시
2. **범죄 이동 과대 추정** — 메타분석 결과 이동은 미미하고, 확산 효과가 더 큼
3. **동기를 무시** — SCP는 "동기를 무시하라"가 아니라 "기회도 봐라"
4. **일반론으로 그침** — "CCTV 더 달자"가 아니라 구체적 범죄 유형별 매칭 필요

## 언제 쓸 것인가

**적합:**
- SARA의 Response 단계에서 개입 옵션 도출
- 특정 범죄 유형의 기회 구조 분석
- CPTED를 넘어서는 비환경적 예방 전략
- 사기·사이버·약물 등 장소 독립적 범죄

**부적합:**
- 동기 분석이 필요한 경우 (Crime Triangle 병행)
- 위기 현장 대응 (ICAT/BCSM 사용)
- 조직범죄 전략 (ILP 사용)

- **Third-Party** — 기법 9(장소 관리자)의 실행
