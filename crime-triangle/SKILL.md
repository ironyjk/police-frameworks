# 범죄 삼각형 — 왜 범죄가 발생하는가

> Background and theory: Read references/foundation.md

---

## SARA의 Analysis 단계에서 사용법

SARA의 A(Analysis) 단계에서 문제를 6개 박스로 분해합니다:

```
1. Offender: 누구인가? 동기는? 이곳을 선택한 이유는?
2. Handler: 가해자를 통제할 수 있는 사람이 존재하는가? 작동하지 않는 이유는?
3. Target: 무엇이 공격당하는가? 매력 요인은? (VIVA: Value, Inertia, Visibility, Access)
4. Guardian: 대상을 보호할 자원이 있는가? 왜 작동하지 않는가?
5. Place: 공간의 어떤 특징이 범죄를 가능하게 하는가?
6. Manager: 장소 관리 주체는 누구인가? 협조 가능한가?
```

**개입은 6개 박스 중 어디를 공략할지 선택하는 문제입니다.**

---

## 개입 옵션 도출법

### Offender 측
- 접근금지 명령, 보호관찰 연계, 재범 방지 프로그램, 교육

### Handler 측
- 가족·보호자 연계, 학교·복지 네트워크, 시설 관리 주체 협력

### Target/Victim 측
- 피해자 교육 (V·I 낮추기), 표적 경화, 보험·보상 체계

### Guardian 측
- 경찰 자원 배치 (최후 수단), 민간 경비, CCTV, 이웃 감시 네트워크

### Place 측
- CPTED 개입, 조명·시야·접근통제, 활용도 증가

### Manager 측
- 건물주·관리소 협약, 지자체 행정 지도, 영업자 책임 강화

## 한국 경찰 적용 예 — 편의점 강도

**문제:** ○○동 24시간 편의점 연쇄 강도, 심야시간 단독 근무 편의점 집중.

| 박스 | 분석 |
|---|---|
| Offender | 20~30대, 단독범, CCTV 회피 모자·마스크. 현금 150만원 이하 소액 현장 선호 |
| Handler | 대부분 노숙·주거불안정 상태 → 개인적 통제자 부재 |
| Target | 심야 현금 보유 편의점, 단독 근무자 |
| Guardian | 단독 근무자는 보호자가 아닌 피해자. 112 신고 외 수단 없음 |
| Place | 이면도로 편의점, 진입·탈출 경로 단순, 조명 제한 |
| Manager | 본사 점주협의회, 개별 점주. 방범 책임은 개별 점주에 분산 |

**개입 옵션:**
1. **Place**: 심야 ATM 현금 상한 조정 (Value 감소)
2. **Manager**: 본사 차원 방범 가이드라인, 2인 근무 인센티브
3. **Guardian**: 편의점 간 긴급 공동대응 네트워크
4. **Offender**: 광역 수사로 연쇄성 입증 후 체포

**단순 순찰 강화는 이 구조를 바꾸지 못함.**

---

## 자주 하는 실수

1. **Offender에만 집착** — 체포만이 답이 아님. 나머지 5개 박스가 열려 있으면 새 가해자가 채움
2. **Target/Place 혼동** — 차를 터는 것은 Target, 주차장 설계는 Place. 다른 개입
3. **Guardian ≠ 경찰** — 경찰은 Guardian의 한 종류일 뿐. 지속가능한 Guardian은 주로 민간
4. **Handler 잊음** — 많은 경우 Handler 활성화가 가장 저렴·효과적

## 언제 쓸 것인가

**적합:**
- SARA의 Analysis 단계
- 반복 문제의 원인 분해
- 경찰 실무자가 "왜 안 줄어드는가" 분석할 때
- 개입 옵션이 순찰 강화에만 머물러 있을 때

**부적합:**
- 단일 사건 수사
- 현장 즉각 대응
- 조직범죄·화이트칼라 등 일상활동이론이 약한 영역
