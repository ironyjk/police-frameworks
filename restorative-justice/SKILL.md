# 회복적 경찰활동 — 처벌이 아니라 회복

> Background and theory: Read references/foundation.md

---

## 언제 쓸 것인가 (★ 경계가 핵심)

**적합:**
- 학교폭력 (경미~중간)
- 이웃 간 분쟁 (소음, 재물 손괴, 경미 폭행)
- 청소년 경미 범죄 (절도, 기물파손)
- 직장 내 갈등으로 인한 고소·고발
- 가해자가 책임을 인정할 의사가 있는 경우
- 피해자가 대화를 원하는 경우 (★ 강요 금지)

**부적합 — 절대 쓰면 안 됨:**
- **성폭력** — 권력 불균형이 극심. 피해자에게 가해자와 대면을 요구하는 것 자체가 2차 가해
- **가정폭력 (심각)** — 통제적 폭력(coercive control)에서는 가해자가 대화를 조종 도구로 사용
- **피해자가 원하지 않는 경우** — 자발성이 전제
- **가해자가 책임을 부인하는 경우** — 대화의 전제가 깨짐
- **조직범죄·반복 폭력범** — 구조적으로 회복 불가능

> **핵심 규칙**: 회복적 접근을 쓸지 말지는 **피해자의 동의**가 출발점입니다.

---

## 한국 경찰 실무 적용

### 학교폭력 사건
1. 접수 시 회복적 접근 적합 여부 판단 (심각도, 반복성, 피해자 의향)
2. 양측 가족에게 회복적 대화 옵션 설명
3. 동의 시 경찰서 여성청소년과 + 학교 + 전문 중재자 연결
4. 대화 결과를 합의서로 문서화
5. 합의 이행 추적 (3개월)

### 이웃 분쟁
1. 112 반복 신고 패턴 확인 (SARA-Scanning)
2. 양측 방문, 회복적 대화 제안
3. 주민자치센터·통장과 협력해 중재 장소 마련
4. 합의 도출 (소음 시간 규정, 사과, 원상복구 등)

### 주의사항
- 경찰관이 직접 중재하지 않음 — **훈련된 중재자**를 연결
- 사건 기록은 정상적으로 작성
- 합의 불성립 시 정상 형사 절차로 복귀
- 피해자에게 "고소 취하하라"는 압박 절대 금지

---

## 자주 하는 실수

1. **피해자 동의 없이 진행** — 회복적 정의의 근본 위반
2. **심각 사건에 적용** — 성폭력·조직폭력에 회복적 접근은 위험
3. **가해자 면죄부** — "사과했으니 끝"이 아님. 실질적 배상·행동 변화 필요
4. **경찰관이 직접 중재** — 전문 훈련 없이 하면 양측 모두 불만
5. **실적으로 전락** — "회복적 처리 건수" 올리기 위해 부적합 사건까지 밀어넣기

## 함께 쓰는 프레임워크

- **SARA** — 반복 분쟁의 근본 원인 분석 후 회복적 접근 결정
- **Procedural Justice** — 회복적 과정 자체의 공정성 유지
- **COP** — 지역사회 자원(중재자, 복지관) 연결
- **Trauma-Informed** — 피해자 트라우마 상태 확인 후 대화 적합 여부 판단
