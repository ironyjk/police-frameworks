# CPTED — 공간이 범죄를 만든다

> Background and theory: Read references/foundation.md

---

## 6대 원칙

### 1. 자연감시 (Natural Surveillance)

**적용:**
- 건물 출입구가 보도·도로에서 보이게
- 1층 상점·창문으로 거리 감시 강화 ("길거리의 눈" — Jane Jacobs)
- 공원 조경: 시야 차단 관목 제거, 키 큰 나무는 가지치기
- 주차장·지하공간: 사각지대 해소
- 가로등: 보행자 얼굴 식별 가능 수준(20m 거리 인상착의 식별)

**실패 패턴:** 높은 담벼락, 창문 없는 긴 외벽, 푹 꺼진 지하보행로

### 2. 접근통제 (Access Control)

**적용:**
- 출입구 최소화, 주동선 명확화
- 저층부 창문·옥상 접근 차단
- 주차장 차단기·카드키
- 조경으로 심리적 경계 형성 (가시 덤불, 단차)

**주의:** 접근통제는 "요새화"가 아닙니다. 정당한 이용자의 동선은 유지하면서 비정당 이용자에게만 저항감을 주는 것이 목표입니다.

### 3. 영역성 강화 (Territorial Reinforcement)

**적용:**
- 공공 공간과 사적 공간의 시각적 구분 (포장 재질 변화, 단차, 식재)
- 진입 지점에 환영/경고 사인
- 주민 커뮤니티 활동 흔적 (화분, 게시판, 벽화)
- 관리 주체 명시 ("관리사무소 ○○호, 연락처 ...")

### 4. 활동지원 (Activity Support)

**적용:**
- 공원 벤치, 운동기구, 어린이 놀이터로 주민 유입
- 상가 야간 운영 유도
- 플리마켓, 지역 행사 개최

**실패 패턴:** 설계만 하고 운영 주체가 없는 공간 → 오히려 무질서 거점화

### 5. 유지관리 (Maintenance)

**적용:**
- 낙서 즉시 제거 ("그래피티 지우기 48시간 규칙")
- 파손된 조명·시설 신속 수리
- 쓰레기 수거 주기 준수
- 빈 건물·빈 부지 관리 주체 명확화

### 6. 표적경화 (Target Hardening)

**적용:** 잠금장치, 방범창, 방탄유리, 볼라드, 디지털 잠금장치

**주의:** 표적경화는 CPTED의 마지막 수단입니다. 먼저 자연감시·접근통제·영역성·활동지원으로 해결하고, 그래도 남는 취약점에 한해 경화합니다.

## 현장 점검 체크리스트

### 자연감시
- [ ] 20m 거리에서 사람 얼굴 식별 가능한 조명인가?
- [ ] 조경이 시야를 가리지 않는가?
- [ ] 1층 창문이 거리를 향해 열려 있는가?
- [ ] 골목 입구에서 끝까지 직선 시야가 확보되는가?

### 접근통제
- [ ] 건물 출입구가 최소화되어 있는가?
- [ ] 저층 창문·옥상 접근 차단되어 있는가?
- [ ] 주차장 차단기·인증 시스템 작동하는가?

### 영역성
- [ ] 공적·사적 공간의 시각적 구분이 명확한가?
- [ ] 관리 주체 표시가 있는가?
- [ ] 주민 활동 흔적이 있는가?

### 활동지원
- [ ] 공간을 실제로 쓰는 합법적 활동이 있는가?
- [ ] 시간대별 이용 계층이 다양한가?

### 유지관리
- [ ] 파손·낙서·쓰레기 방치 기간이 얼마인가?
- [ ] 조명 고장 수리 체계가 있는가?

### 표적경화
- [ ] 필수 잠금·방범 설비가 작동하는가?
- [ ] 고가치 대상에 적절한 보호가 있는가?

## 자주 하는 실수

1. **하드웨어만** — CCTV 설치 = CPTED로 오해. 2세대 요소 누락
2. **표적경화 우선** — 요새화 + 황폐화 경로
3. **주민 참여 없음** — 영역성·활동지원은 주민 없이 불가
4. **유지관리 예산 누락** — 설치 예산만 있고 관리 예산 없음 → 2년 뒤 원상복구
5. **범죄 이동 무시** — 한 블록만 개선하면 옆 블록으로 이동

## 언제 쓸 것인가

**적합:**
- 특정 공간(주차장, 공원, 골목, 학교)의 반복 범죄
- 신규 개발 계획 심의
- 경찰 범죄예방진단팀의 환경 점검
- SARA의 Response 단계에서 환경 개입 설계

**부적합:**
- 사람 중심 문제 (조사, 위기 대응)
- 데이터 기반 배치 문제 (Hot Spots의 영역)
- 사이버 범죄
