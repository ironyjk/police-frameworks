# ICAT — 물리력보다 시간을

> Background and theory: Read references/foundation.md

PERF의 디에스컬레이션 훈련 프로그램. Communications + Assessment + Tactics를 통합하여 비무기 저항·정신건강 위기 상황에서 물리력 사용을 줄입니다.

---

## 3대 구성요소

### 1. Communications (의사소통)

**목표:** 언어적·비언어적 의사소통으로 긴장을 낮춤

**도구:**
- BCSM의 Active Listening Skills
- 개방형 질문
- 감정 명명
- 침묵과 공간
- 존엄한 호칭

**금기:**
- 고함, 욕설
- 여러 명이 동시에 소리
- 모욕적 명령
- 비웃음, 조롱

### 2. Assessment (평가)

**목표:** 상황의 지속적·동적 평가

**도구:** Critical Decision-Making Model (CDMM)
- 정보 수집 → 위협 평가 → 권한·정책 → 목표 → 옵션 → 행동 → 검토
- 1회성이 아닌 **계속 반복**

### 3. Tactics (전술)

**목표:** 물리적 환경을 활용해 시간·거리·엄폐 확보

**원칙:**
- Time (시간) — 서두르지 않기
- Distance (거리) — 물러서기
- Cover (엄폐) — 물리적 보호
- Team (팀) — 역할 분담

---

## 실행 프로토콜: CDMM 6단계

1. **정보 수집** — 신고 내용, 현장 관찰, 정신건강·약물·무기 단서. 계속 업데이트.
2. **위협 평가** — 즉각 위협 vs 지연 가능 구분. "칼 들고 있지만 10m 거리, 자해만" ≠ "칼 들고 돌진"
3. **권한·정책 검토** — 법적 권한, 기관 정책, 물리력 합법성
4. **목표 설정** — 최우선: 모든 당사자 안전 → 상황 안정화 → 문제 해결. 체포는 수단이지 목적이 아님
5. **옵션 파악** — 대화, 후퇴, 엄폐, 대기, 지원 요청, 정신건강 전문가 연락. 물리력은 마지막
6. **행동·재검토** — 실행 → 즉시 결과 관찰 → 효과 없으면 1번으로 복귀

**핵심: 결정은 한 번이 아니라 수십 번 반복된다.**

## 시간·거리·엄폐 (TDC) 실행

| 원칙 | 판단 질문 | 행동 |
|------|----------|------|
| Time | "이 사람이 지금 누군가를 해치고 있는가?" 아니오면 → | 시간 만들기 |
| Distance | 근접해 있는가? → | 물러서기 (패배 아닌 전술) |
| Cover | 물리적 보호물이 있는가? → | 벽·차량·기둥 뒤로 |

**TDC 확보 시:** 물리력 압박 감소, 대화 공간 생성, 팀 도착 대기, 당사자 진정

## De-escalation 기법

### 언어적
- 낮고 차분한 목소리, 천천히, 짧은 문장
- 이름 사용 ("OO씨, 제 이름은...")
- 개방형 질문 ("무슨 일이 있으셨어요?")
- 감정 인정 ("많이 힘드신 것 같네요")

### 하지 말 것
- "진정해!", "명령이다!", "안 그러면...", "당신이 문제야"
- 여러 명 동시 말하기

### 비언어적
- 손 보이기, 무기 손 내리기
- 45도 각도 (정면 대결 피함)
- 거리 유지, 갑작스러운 움직임 금지
- 표정 부드럽게

### 팀 구성
- **Contact officer** — 주 대화자 (한 사람만)
- **Cover officer** — 주변 안전 확보
- **Supervisor** — 전략적 의사결정
- **Additional** — 의료·가족·통역

→ 한 명만 말하는 것이 핵심. 여러 명이 동시에 말하면 혼란 가중.

## 자주 하는 실수

1. **"디에스컬레이션 = 약함"으로 오해** — 실증 결과는 정반대: 경찰관 부상 감소
2. **상황 급히 해결 시도** — 시간이 가장 강력한 동맹
3. **정신건강 위기를 범죄로 프레임** — 전혀 다른 접근 필요
4. **여러 명 동시 명령** — 당사자 인지 과부하 → 폭력 위험 증가
5. **엄폐 대신 대면** — TDC 원칙 위반
6. **CDMM 일회성 적용** — 계속 반복되어야 함

## 언제 쓸 것인가

**적합:**
- 정신건강 위기 신고
- 비무기·경무기 저항
- 격앙된 시민 대응
- 자살 시도 현장 (+ BCSM)
- 배회·방황·혼동 상태

**부적합:**
- 즉각 생명 위협 (능동적 총격, 현행 폭력)
- 조직범죄 급습
- 일상적 교통 단속 (단, 원칙은 여전히 유용)
