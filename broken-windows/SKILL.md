# Broken Windows — 영향력이 가장 큰, 그리고 가장 오용된 이론

> Background and theory: Read references/foundation.md

---

## 경찰 실무자 주의사항

지구대·경찰서가 **무질서·경범죄 관련 민원**을 다룰 때 주의할 것:

### 위험 신호

민원이 다음 형태로 올 때 Broken Windows의 잘못된 적용에 빠지기 쉽습니다:

- "노숙인 때문에 동네 이미지가 나빠져요"
- "외국인 노동자가 밤에 떼 지어 다녀서 불안해요"
- "청소년들이 공원에서 시끄러워요"
- "ㅇㅇ 업소 때문에 분위기가 나빠요"

**공통점:** 특정 **집단**에 대한 불쾌감·불안을 범죄 문제로 프레임.

### 경찰 실무 체크리스트

무질서 관련 민원·정책을 다룰 때:
- [ ] 문제가 **환경**인가 **사람**인가? 환경이라면 CPTED/행정, 사람이면 복지·의료 연계 우선 검토
- [ ] 제안된 조치가 **특정 집단**에 차별적 영향을 주는가?
- [ ] 절차적 정의 4요소에 부합하는가?
- [ ] 근본 원인(빈곤, 정신건강, 중독)에 접근하는가, 표면만 다루는가?
- [ ] "눈에 안 보이게" 쫓아내는 것과 "문제 해결"을 구분했는가?
- [ ] 경찰 외 주체(복지관, 구청, 의료)를 충분히 활용했는가?

### 권장 언어 전환

| 피할 표현 | 권장 표현 |
|---|---|
| 엄격한 단속 | 체계적 환경 유지 + 복지 연계 |
| 무관용 | 지역 공동 책임 |
| 문제 인물 제거 | 취약집단 지원 + 경계선 관리 |
| 경범죄 검거 | 원인 분석 기반 개입 |

---

## 자주 하는 실수

1. **원이론 그대로 적용** — 현대 연구가 비판하는 방식
2. **Zero Tolerance와 COP 혼동** — 철학이 정반대
3. **"무질서 = 범죄"로 등식** — 인과 아니라 상관일 수 있음
4. **특정 집단 표적** — 법적·윤리적 문제
5. **복지·의료 연계 누락** — 근본 원인 무시
6. **단기 효과 과신** — 장기 부작용 무시

---

## 언제 쓸 것인가

**적합:**
- 무질서·경범죄 민원 분석 (**비판적 시각으로**)
- 주민 측에서 "엄격한 단속" 요구가 들어왔을 때 사전 검토
- Broken Windows 관련 정책 제안 검증

**부적합 (이것은 기본 도구가 아니라 경계 도구입니다):**
- 긍정적 액션 프레임워크로 사용
- 단독 문제해결 도구
- 범죄 예방 기본 전략

**추천 활용법:** 다른 프레임워크와 함께 사용해, 집중 단속식 해석이 등장할 때 브레이크 역할.

## 함께 쓰는 프레임워크

- **CPTED** — 환경 유지의 올바른 형태
- **COP** — 주민 파트너십
- **SARA** — 문제 중심 분석
- **Procedural Justice** — 부작용 방지
