# NICHD Protocol — Background & Theory

> "Children can provide accurate testimony — if adults ask the right questions in the right way." — Michael Lamb

## 배경

아동은 성인과 다른 방식으로 기억하고 표현합니다:
- **피암시성(suggestibility)** 이 높음 — 유도 질문에 취약
- 어른의 기대에 맞추려는 경향 — "네"라고 대답하는 편향
- 시간·빈도·순서 개념이 미발달
- 감정 언어가 제한적
- 반복 질문을 "내 대답이 틀렸나 보다"로 해석

1990년대 Michael Lamb(NICHD)은 기존 아동면담의 문제점을 체계적으로 연구하고, **아동 발달 수준에 맞춘 구조화된 면담 프로토콜**을 개발했습니다.

**핵심 원칙**: 아동에게서 정확한 정보를 끌어내려면 **개방형 질문을 최대한 많이, 유도형 질문을 최소한**으로 사용해야 합니다.

## 성인 인지면담(Cognitive Interview)과의 차이

| | Cognitive Interview (성인) | NICHD Protocol (아동) |
|---|---|---|
| **대상** | 성인 피해자·목격자 | 3~14세 아동 |
| **맥락 복원** | "눈 감고 당시로 돌아가세요" | 사용 제한 — 아동은 상상과 혼동 가능 |
| **역순 회상** | 적극 활용 | 사용 안 함 — 인지 부하 과다 |
| **관점 전환** | 신중히 사용 | 사용 안 함 — 상상과 기억 혼동 |
| **자유 진술** | 핵심 (둘 다 동일) | 핵심 (둘 다 동일) — 단, 더 많은 지원 필요 |
| **질문 유형** | 개방형 우선 | 개방형 극도로 우선 — 유도형 최소화 |

## 아동 연령별 조정

### 3~5세 (미취학)
- 문장 극도로 짧게, 단순한 단어
- "언제"보다 "뭘 하고 있었어?" (시간 개념 미발달)
- 인형·그림 보조 도구 신중 사용 (오히려 오염 가능)
- 집중 시간 15~20분이 한계

### 6~9세 (초등 저학년)
- 자유 회상 가능하나 짧음 → "그다음에는?" 반복
- 시간·순서 개념 발달 중
- 비밀 약속·위협에 대한 두려움 확인

### 10~14세 (초등 고학년~중학생)
- 성인에 가까운 면담 가능하나 여전히 피암시성 주의
- 또래 압력·수치심 고려
- 디지털 증거(채팅, SNS) 연계 질문

## 한국 법령·연구 맥락

### 관련 법령
- **성폭력처벌법 제30조** (19세미만피해자등 진술 내용 등의 영상녹화 및 보존 등) — 검사·사법경찰관은 19세 미만 피해자 진술을 **의무적으로** 영상녹화. 조사 전 피해자 나이·발달 단계·심리 상태·장애 정도를 고려한 방식으로 영상녹화 사실과 증거 사용 가능성을 설명하여야 함. 피해자·법정대리인이 원하지 않으면 녹화 금지.
- **아동학대처벌특례법 제17조** (준용) — 아동학대범죄 조사·심리에 성폭력처벌법 제29조~제32조·제34조~제41조를 준용. 13세 미만 또는 의사소통 장애 아동이 참고인·증인인 경우 진술조력인 제도(성폭력처벌법 제36조~제39조) 준용.

### 국내 연구·제도
- **해바라기센터** (전국 39개소, 2025 기준) — 19세 미만 성폭력 피해자·장애인 피해자 전담. NICHD 프로토콜 준수 훈련 이수 조사관 배치.
- **아동보호전문기관** (2023년 기준 전국 71개소, 보건복지부 소관·지자체 위탁 운영) — 아동학대 전담, 경찰 수사와 병행.
- **법무부 진술조력인 제도** — 성폭력처벌법 전부개정(2012.12.19)으로 도입, **2013.12.19 시행**. 19세 미만 피해자·장애인 피해자의 수사·재판 과정에 전문 진술조력인 참여. NICHD 프로토콜 이해가 조력의 전제.
- **법무부 (2020) 『아동·장애인 진술 조력인 운영 매뉴얼』** — NICHD 프로토콜 한국 실무 적용 지침.

### 실무 유의
- 한국에서 아동 피해자 조사는 **해바라기센터 의뢰가 원칙**입니다. 지구대·경찰서 일반 조사실에서 아동을 직접 조사하면 법원에서 진술 신빙성 의심을 받을 수 있습니다.
- 진술 영상녹화 후 원본은 법정에 증거로 제출되며, **반복 진술 요구 금지** 원칙이 법적으로도 존중됩니다.
