# NICHD 프로토콜 — 아이의 말을 제대로 듣는 법

> Background and theory: Read references/foundation.md

---

## NICHD 면담 구조 (7단계)

### 1단계: 소개 (Introduction)
- 면담자 자기소개, 면담 목적 간단 설명
- **진실 말하기 규칙** 설정: "오늘 여기서는 진짜 있었던 일만 이야기해요"
- "모른다고 해도 괜찮아요" 명시
- "제가 틀리게 말하면 고쳐줘도 돼요" 명시
- 녹화 동의 (법정대리인 + 아동 본인)

### 2단계: 라포 형성 (Rapport-Building)
- 아동이 편안해질 때까지 중립적 주제로 대화 (좋아하는 것, 학교, 친구)
- **목적**: 면담 환경 안전감 + 아동의 언어 수준 파악

### 3단계: 자유 회상 훈련 (Episodic Memory Training)
- "어제 학교 끝나고 집에 올 때까지 있었던 일을 다 이야기해 줄래?"
- 중립적 사건에 대해 자유 회상 연습
- **목적**: "여기서는 길고 자세하게 이야기하는 거야"를 학습시킴

### 4단계: 사건 전환 (Transition to Substantive Issues)
- "오늘 여기 온 이유가 뭔지 알아?" (개방형)
- 아동이 스스로 사건을 언급하도록 유도
- 아동이 자발적으로 말하지 않으면 점진적으로 구체화

### 5단계: 자유 회상 — 사건 본체 (Free Recall)
- "무슨 일이 있었는지 처음부터 끝까지 다 이야기해 줄래?"
- **방해 금지** — 아동이 완전히 끝낼 때까지
- 아동의 단어를 그대로 사용해 재질문

### 6단계: 명료화 질문 (Clarification)
개방형 → 지시형 → 선택형 → 유도형 순서로 (★ 가능한 한 개방형에 머물기)

| 질문 유형 | 예시 | 정보 가치 | 오염 위험 |
|---|---|---|---|
| **개방형 (Recall)** | "그 부분 더 이야기해 줄래?" | ★★★ 높음 | ★ 낮음 |
| **지시형 (Directed)** | "그 사람 얼굴이 어떻게 생겼어?" | ★★ 중간 | ★★ 중간 |
| **선택형 (Option)** | "낮이었어, 밤이었어?" | ★ 낮음 | ★★★ 높음 |
| **유도형 (Suggestive)** | "아저씨가 때렸지?" | ✕ 사용 금지 | ✕✕✕ 극히 위험 |

### 7단계: 종결 (Closure)
- 중립적 주제로 전환
- "더 말하고 싶은 거 있니?"
- 감사 표현, 앞으로 일어날 일 간략 설명
- 불안해하면 지원자(보호자·상담사) 연결

## 한국 적용

### 현행 제도
- **해바라기센터**: 아동 성폭력·학대 전담 — 영상녹화 면담실 운영
- **아동보호전문기관**: 학대 신고 조사
- **진술 영상녹화**: 형사소송법 제221조의2
- **신뢰관계인 동석**: 아동 면담 시 보호자·전문가 동석 가능

### 개선 기회
- 해바라기센터 면담관의 NICHD 프로토콜 준수율 편차가 큼
- 경찰 초동 대응 시 아동 접촉 방법 교육 필요
- **Trauma-Informed + NICHD 결합**: 아동의 트라우마 반응을 이해하면서 비유도적 면담

## ★ 가장 효과적인 단일 개입

면담 전에 **자유 회상 훈련(3단계)**을 반드시 실행하는 것. 이 훈련 없이 바로 사건 질문에 들어가면 아동은 짧은 답변만 하고, 면담자는 구체적 질문을 쏟아내게 되어 유도의 악순환이 시작됩니다. 5분의 훈련이 이후 30분의 면담 품질을 결정합니다.

## 자주 하는 실수

1. **유도 질문** — "아저씨가 때렸지?"는 증거 오염. 법정에서 진술 전체가 배제될 수 있음
2. **같은 질문 반복** — 아동은 "내 답이 틀렸구나"로 인식하고 답변을 바꿈
3. **자유 회상 건너뛰기** — 바로 구체적 질문 → 아동이 수동적으로 변함
4. **보상·칭찬 남용** — "잘했어!"가 특정 답변을 강화시킴. 격려는 내용 중립적으로
5. **어른의 언어 강요** — 아동이 "거기"라고 하면 "거기"로 후속. "성기"로 바꾸지 않기

## 함께 쓰는 프레임워크

- **Trauma-Informed** — 아동 트라우마 반응 이해 (먼저 안정화)
- **PEACE** — 전체 면담 관리 틀 (단, A 단계를 NICHD 방식으로 대체)
- **Repeat Victimization** — 아동학대는 본질적으로 반복범죄
- **Risk Assessment** — 가정 내 학대 위험 수준 평가
