# NICHD 프로토콜 — 아이의 말을 제대로 듣는 법

> "Children can provide accurate testimony — if adults ask the right questions in the right way." — Michael Lamb

## 배경

아동은 성인과 다른 방식으로 기억하고 표현합니다:
- **피암시성(suggestibility)** 이 높음 — 유도 질문에 취약
- 어른의 기대에 맞추려는 경향 — "네"라고 대답하는 편향
- 시간·빈도·순서 개념이 미발달
- 감정 언어가 제한적
- 반복 질문을 "내 대답이 틀렸나 보다"로 해석

1990년대 Michael Lamb(NICHD)은 기존 아동면담의 문제점을 체계적으로 연구하고, **아동 발달 수준에 맞춘 구조화된 면담 프로토콜**을 개발했습니다.

**핵심 원칙**: 아동에게서 정확한 정보를 끌어내려면 **개방형 질문을 최대한 많이, 유도형 질문을 최소한**으로 사용해야 합니다.

---

## 성인 인지면담(Cognitive Interview)과의 차이

| | Cognitive Interview (성인) | NICHD Protocol (아동) |
|---|---|---|
| **대상** | 성인 피해자·목격자 | 3~14세 아동 |
| **맥락 복원** | "눈 감고 당시로 돌아가세요" | 사용 제한 — 아동은 상상과 혼동 가능 |
| **역순 회상** | 적극 활용 | 사용 안 함 — 인지 부하 과다 |
| **관점 전환** | 신중히 사용 | 사용 안 함 — 상상과 기억 혼동 |
| **자유 진술** | 핵심 (둘 다 동일) | 핵심 (둘 다 동일) — 단, 더 많은 지원 필요 |
| **질문 유형** | 개방형 우선 | 개방형 극도로 우선 — 유도형 최소화 |

## NICHD 면담 구조 (7단계)

### 1단계: 소개 (Introduction)
- 면담자 자기소개, 면담 목적 간단 설명
- **진실 말하기 규칙** 설정: "오늘 여기서는 진짜 있었던 일만 이야기해요"
- "모른다고 해도 괜찮아요" 명시
- "제가 틀리게 말하면 고쳐줘도 돼요" 명시
- 녹화 동의 (법정대리인 + 아동 본인)

### 2단계: 라포 형성 (Rapport-Building)
- 아동이 편안해질 때까지 중립적 주제로 대화 (좋아하는 것, 학교, 친구)
- **목적**: 면담 환경 안전감 + 아동의 언어 수준 파악
- 아동의 자발적 말하기 패턴 관찰 (문장 길이, 어휘, 시간 표현)

### 3단계: 자유 회상 훈련 (Episodic Memory Training)
- "어제 학교 끝나고 집에 올 때까지 있었던 일을 다 이야기해 줄래?"
- 중립적 사건(사건과 무관)에 대해 자유 회상 연습
- **목적**: 아동에게 "여기서는 길고 자세하게 이야기하는 거야"를 학습시킴
- 아동이 짧게 답하면: "그다음에는?"으로 더 끌어내기 연습

### 4단계: 사건 전환 (Transition to Substantive Issues)
- "오늘 여기 온 이유가 뭔지 알아?" (개방형)
- 아동이 스스로 사건을 언급하도록 유도
- 아동이 자발적으로 말하지 않으면 점진적으로 구체화:
  - "누가 네 이야기를 들어야 한다고 했는지 말해줄래?"
  - 최후: "선생님이 네 몸에 대해 걱정이 된다고 하셨는데..."

### 5단계: 자유 회상 — 사건 본체 (Free Recall)
- "무슨 일이 있었는지 처음부터 끝까지 다 이야기해 줄래?"
- **방해 금지** — 아동이 완전히 끝낼 때까지
- 개방형 후속: "그리고?", "그다음에는?", "더 있니?"
- 아동의 단어를 그대로 사용해 재질문 (아동의 언어를 면담자의 언어로 바꾸지 않기)

### 6단계: 명료화 질문 (Clarification)
개방형 → 지시형 → 선택형 → 유도형 순서로 (★ 가능한 한 개방형에 머물기)

| 질문 유형 | 예시 | 정보 가치 | 오염 위험 |
|---|---|---|---|
| **개방형 (Recall)** | "그 부분 더 이야기해 줄래?" | ★★★ 높음 | ★ 낮음 |
| **지시형 (Directed)** | "그 사람 얼굴이 어떻게 생겼어?" | ★★ 중간 | ★★ 중간 |
| **선택형 (Option)** | "낮이었어, 밤이었어?" | ★ 낮음 | ★★★ 높음 |
| **유도형 (Suggestive)** | "아저씨가 때렸지?" | ✕ 사용 금지 | ✕✕✕ 극히 위험 |

### 7단계: 종결 (Closure)
- 중립적 주제로 전환 (다시 좋아하는 것 이야기)
- "더 말하고 싶은 거 있니?"
- 감사 표현
- 앞으로 일어날 일 간략 설명
- 아동이 불안해하면 지원자(보호자·상담사) 연결

## 한국 적용

### 현행 제도
- **해바라기센터**: 아동 성폭력·학대 전담 — 영상녹화 면담실 운영
- **아동보호전문기관**: 학대 신고 조사
- **진술 영상녹화**: 형사소송법 제221조의2 — 아동·장애인 피해자 진술 영상 녹화
- **신뢰관계인 동석**: 아동 면담 시 보호자·전문가 동석 가능

### 개선 기회
- 해바라기센터 면담관의 NICHD 프로토콜 준수율 편차가 큼
- 경찰 초동 대응 시 아동 접촉 방법 교육 필요 (첫 접촉이 이후 면담 정확성에 영향)
- **Trauma-Informed + NICHD 결합**: 아동의 트라우마 반응을 이해하면서 비유도적 면담

## 아동 연령별 조정

### 3~5세 (미취학)
- 문장 극도로 짧게, 단순한 단어
- "언제"보다 "뭘 하고 있었어?" (시간 개념 미발달)
- 인형·그림 보조 도구 신중 사용 (오히려 오염 가능)
- 집중 시간 15~20분이 한계

### 6~9세 (초등 저학년)
- 자유 회상 가능하나 짧음 → "그다음에는?" 반복
- 시간·순서 개념 발달 중
- 비밀 약속·위협에 대한 두려움 확인

### 10~14세 (초등 고학년~중학생)
- 성인에 가까운 면담 가능하나 여전히 피암시성 주의
- 또래 압력·수치심 고려
- 디지털 증거(채팅, SNS) 연계 질문

---

## ★ 가장 효과적인 단일 개입

면담 전에 **자유 회상 훈련(3단계)**을 반드시 실행하는 것. 이유: 이 훈련 없이 바로 사건 질문에 들어가면 아동은 짧은 답변만 하고, 면담자는 구체적 질문을 쏟아내게 되어 유도의 악순환이 시작됩니다. 5분의 훈련이 이후 30분의 면담 품질을 결정합니다.

---

## 자주 하는 실수

1. **유도 질문** — "아저씨가 때렸지?"는 증거 오염. 법정에서 진술 전체가 배제될 수 있음
2. **같은 질문 반복** — 아동은 "내 답이 틀렸구나"로 인식하고 답변을 바꿈
3. **자유 회상 건너뛰기** — 바로 구체적 질문 → 아동이 수동적으로 변함
4. **보상·칭찬 남용** — "잘했어!"가 특정 답변을 강화시킴. 격려는 내용 중립적으로
5. **어른의 언어 강요** — 아동이 "거기"라고 하면 "거기"로 후속. "성기"로 바꾸지 않기

## 함께 쓰는 프레임워크

- **Trauma-Informed** — 아동 트라우마 반응 이해 (먼저 안정화)
- **PEACE** — 전체 면담 관리 틀 (단, A 단계를 NICHD 방식으로 대체)
- **Repeat Victimization** — 아동학대는 본질적으로 반복범죄
- **Risk Assessment** — 가정 내 학대 위험 수준 평가

## 한국 법령·연구 맥락

### 관련 법령
- **성폭력처벌법 제30조** (19세 미만 피해자 영상녹화·보존) — 검사·사법경찰관은 19세 미만 피해자 진술을 **반드시** 영상녹화. 피해자 나이·발달 단계·심리 상태·장애 정도를 고려한 조사 방식 설명 의무.
- **헌재 2021.12.23. 2018헌바524** — 영상녹화물 증거능력 특례 일부 위헌. 2023년 개정으로 피고인 반대신문권 보장 강화 + 피해자 2차 피해 방지 균형.
- **아동학대처벌특례법 제17조** — 아동학대 피해 아동 진술 시 신뢰관계인 동석, 영상녹화.
- **형사소송법 제221조의2** (영상녹화물) — 아동·장애인 영상녹화물의 증거능력 특례.

### 국내 연구·제도
- **해바라기센터** (전국 39개소, 2025 기준) — 19세 미만 성폭력 피해자·장애인 피해자 전담. NICHD 프로토콜 준수 훈련 이수 조사관 배치.
- **아동보호전문기관** (전국 71개소) — 아동학대 전담, 경찰 수사와 병행.
- **법무부 진술조력인 제도** (2013~) — 13세 미만·장애인 피해자의 진술에 전문가 동석. NICHD 프로토콜 이해가 조력의 전제.
- **법무부 (2020) 『아동·장애인 진술 조력인 운영 매뉴얼』** — NICHD 프로토콜 한국 실무 적용 지침.

### 실무 유의
- 한국에서 아동 피해자 조사는 **해바라기센터 의뢰가 원칙**입니다. 지구대·경찰서 일반 조사실에서 아동을 직접 조사하면 법원에서 진술 신빙성 의심을 받을 수 있습니다.
- 진술 영상녹화 후 원본은 법정에 증거로 제출되며, **반복 진술 요구 금지** 원칙이 법적으로도 존중됩니다.
